# About 페이지 배경 사진 추가 — 설계

**날짜:** 2026-06-17
**목적:** About 페이지에 이 화제가 어떻게 정치 이슈가 됐는지 보여주는 보도 맥락 사진 1장을 중립적으로 추가한다.

## 배경

2026년 6·3 지방선거에서 서로 다른 투표소의 동일 득표(쌍둥이 득표)가 정치권에서 화제가 됐고, 이 도구는 그 빈도를 보여주는 중립적 맥락 자료다. About 페이지("데이터 출처와 계산 방법")는 이미 도구의 취지와 중립성을 설명하는 페이지이므로, 이 화제의 배경을 보여주는 사진을 넣기에 가장 자연스러운 자리다.

## 결정 사항

- **배치:** About.tsx 상단 인트로 문단(중립성 고지) 바로 아래, 첫 Section("원본 데이터 출처") 앞에 새 `Section title="배경"`.
- **사진:** `~/Downloads/4a71f8e32afa5a720c478aae47b790d4.png` → `web/public/about-background.png`로 복사. `/about-background.png`로 참조.
- **구조:** `<figure>` + `<img>` + `<figcaption>`. 기존 카드 스타일(`--color-surface`, `--color-border`, `rounded-lg`)과 통일. 이미지는 `w-full`.
- **캡션 (중립 + 출처):**
  > 장동혁 국민의힘 대표가 2026년 6월 서로 다른 투표소의 동일 득표를 두고 조작 가능성을 제기하는 모습. 이 도구는 이런 주장의 옳고 그름을 판단하지 않고, 과거 선거에서 같은 현상이 얼마나 자주 있었는지만 보여준다.
  > *출처: TJB, 2026-06-09 (기사 링크)*
- **출처:** TJB(대전방송), 2026-06-09, https://www.tjb.co.kr/news05/bodo/view/id/99311/version/1

## 중립성 가드

캡션은 "장동혁이 옳다/그르다"를 말하지 않고 "이런 주장이 제기됐다"는 사실만 서술한다. 도구의 취지(판단 안 함, 빈도만)를 사진 옆에서 재확인한다. About 인트로의 기존 중립 고지문과 일관.

## 범위 밖 (YAGNI)

- 히어로/메인 화면 변경 없음
- 이미지 라이트박스·캐러셀 없음
- 반응형은 기존 `max-w-3xl` 컨테이너 + `w-full`로만

`cd web && npm run build` (tsc -b && vite build) 통과.
