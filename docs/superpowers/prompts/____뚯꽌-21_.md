## 먼저: 대선용 스키마/검증을 지선과 분리할지 결정

지선 파이프라인은 `etl/local/`에 있고 schema COLUMNS에 `선거구명`이 있다. 대선은 선거구가 없고 대신 전국 단위다. 다음 방침을 따르라:

- 대선 파서·스키마·검증·빌드는 **`etl/pres/`** 에 새로 만든다(지선 `etl/local/`을 건드리지 말 것). 단 **schema와 validate는 지선 것을 최대한 재사용**하라 — `etl/local/schema.py`의 COLUMNS를 그대로 쓰되, 대선엔 `선거구명`이 항상 빈 문자열이면 된다(컬럼 자체는 유지해 동일 tidy 포맷). `etl/local/validate.py`의 `run_all_checks`도 그대로 재사용한다(시도별 후보 일관성 검사는 대선에선 전국 단일 후보라 항상 통과 — 무해). 즉 `etl/pres/parse_21st.py`는 `from etl.local.schema import COLUMNS, to_dataframe, normalize_level`, `from etl.local.validate import run_all_checks`를 import.
- `etl/local/official_totals.py`의 OFFICIAL_TOP는 `(회차, 선거종류, 시도)` 키다. 대선은 `선거종류="대통령"`으로 추가하되, **대선 회차가 지선 회차(3~8)와 겹치므로** 키 충돌을 피해야 한다. 별도 딕셔너리 `OFFICIAL_TOP_PRES`를 `etl/pres/official_pres.py`에 새로 만들고 키를 `(대, 시도)` 또는 `(21, "대통령", 시도)`로 두라(지선과 안 겹치게 선거종류를 키에 포함). 네 판단으로 깔끔하게.

## 원본 구조 (실측 완료 — 검증하되 신뢰해도 됨)

파일: `data_raw/제21대_대통령선거_개표결과.xlsx` (단일 .xlsx, 시트 'sheet1')

- r0~2: 제목/빈 행
- r3: 컬럼 라벨 `['시도명','구시군명','읍면동명','투표구명','선거인수','투표수','후보자별 득표수',...]`
  → col0=시도명, col1=구시군명, col2=읍면동명, col3=투표구명, col4=선거인수, col5=투표수, col6+=후보별 득표, 마지막에 '계'
- r4: 후보 헤더 `['','','','','','','더불어민주당\n이재명','국민의힘\n김문수','개혁신당\n이준석','민주노동당\n권영국','무소속\n송진호','계']`. `정당\n이름` 형식. 끝의 '계'는 유효표 합(후보 아님) — 제외.
- r5: 빈 행
- r6+: 데이터. 4단 계층:
  - r6 `전국 | 합계` → 전국 총계. SKIP(시도도 구시군도 아님).
  - r7 `서울특별시 | 합계(특별시)` → **시도 합계**. SKIP(구시군 단위가 아님). '합계(특별시)' 처럼 괄호가 붙으니 '합계'로 시작하면 시도합계로 보고 skip.
  - r8 `· | 종로구 | 합계` → **구시군 합계** → totals로. (시도명은 빈칸이면 직전 값 carry-forward)
  - r9~11 `· | · | 거소·선상투표 / 관외사전투표 / 재외투표` → 구시군 단위 특수투표. emit. level: 거소·선상투표→거소선상, 관외사전투표→관외사전투표, **재외투표→재외투표**(대선 특유, normalize_level이 그대로 통과시킴 — 새 level 값).
  - r12 `· | · | 청운효자동 | 소계` → 읍면동 소계. SKIP(하위 합).
  - r13 `· | · | · | 관내사전투표` → 그 읍면동의 관내사전. emit. level=사전투표(=관내사전투표를 normalize). 읍면동=청운효자동(직전 carry-forward).
  - r14~ `· | · | · | 청운효자동제1투` → 개별 투표구(당일투표). emit. level=당일투표.

## 핵심 결정: 쌍둥이 단위 = 읍면동 (투표구 합산)

송도 논란은 **읍면동 단위 관내사전투표**가 일치한 것이다(송도1동 vs 송도2동). 따라서:
- **관내사전투표**: 읍면동 단위로 1행 emit(level=사전투표, 읍면동=청운효자동). 이미 원본이 읍면동당 1개 관내사전투표 행을 주므로 그대로 emit.
- **당일투표(개별 투표구)**: 같은 읍면동의 여러 투표구(제1투/제2투/...)를 **읍면동 단위로 합산해 1행** emit(level=당일투표, 읍면동=청운효자동). 투표구를 개별로 두면 쌍둥이 단위가 너무 잘게 쪼개져 송도 맥락과 안 맞는다. → 읍면동+level로 group해서 득표·투표수·선거인수를 합산.
  - 단 합계 정합을 위해: 구시군 합계 = 거소선상 + 관외사전 + 재외 + Σ(읍면동별 [관내사전 + 당일합산]). 검증이 이걸 확인하도록 totals를 구시군 합계행 투표수로 담아라.
- `투표구명` 자체는 tidy row에 별도로 남기지 않는다(읍면동으로 집계하므로). COLUMNS에 투표구 컬럼이 없으니 선거구명=빈문자열, 읍면동=읍면동명.

## level 정규화

`normalize_level`은 `관내사전투표→사전투표, 관외사전투표→관외사전투표, 거소·선상투표→거소선상`을 이미 매핑한다. `재외투표`는 미매핑이라 그대로 '재외투표'로 통과 — 그게 맞다(대선 특유 채널). `소계`·`합계`·`합계(특별시)`·`전국`은 emit 안 함.

## official 검증 (위키 대조)

21대 대선(2025) 시도지사가 아니라 **대통령**이다. 1위는 전국 이재명. 시도별 1위 득표를 3~4개 시도만 위키백과 "제21대 대통령선거" 전용 결과표로 검증해 OFFICIAL_TOP_PRES에 추가하라. 예: 서울 이재명/김문수 중 1위, 경기 이재명 등. 전국 이재명 총득표(17,287,513 — r6의 col6)도 참고. **네 파서 합 == 위키 == 원본 합계행**이어야 한다. 출처 URL과 득표율 교차검증을 보고하라.

## 테스트 파일 (verbatim 작성)

`tests/pres/__init__.py`(빈 파일)과 `tests/pres/test_parse_21st.py`:
```python
from etl.pres.parse_21st import parse_21st
from etl.local.schema import to_dataframe, COLUMNS
from etl.local.validate import run_all_checks
from etl.pres.official_pres import OFFICIAL_TOP_PRES

PATH = "data_raw/제21대_대통령선거_개표결과.xlsx"


def test_parse_21st_returns_rows():
    rows, totals = parse_21st(PATH)
    assert len(rows) > 10000
    df = to_dataframe(rows)
    assert list(df.columns) == COLUMNS


def test_parse_21st_no_subtotal_leak():
    """소계·합계가 데이터로 새지 않았는지 — 읍면동에 '소계'/'합계' 없어야."""
    rows, _ = parse_21st(PATH)
    df = to_dataframe(rows)
    assert not df["읍면동"].astype(str).str.contains("소계|합계|전국", regex=True).any()


def test_parse_21st_has_sajeon():
    """관내사전투표(=사전투표) 행이 읍면동 단위로 있어야 — 송도 맥락."""
    rows, _ = parse_21st(PATH)
    df = to_dataframe(rows)
    assert (df["level"] == "사전투표").sum() > 3000


def test_parse_21st_passes_all_validation():
    rows, totals = parse_21st(PATH)
    for row in rows:
        row.setdefault("선거일", "2025-06-03")
    df = to_dataframe(rows)
    errors = run_all_checks(df, totals, OFFICIAL_TOP_PRES)
    assert errors == [], "검증 위반:\n" + "\n".join(f"  {n}: {d}" for n, d in errors[:20])
```
